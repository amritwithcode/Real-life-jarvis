# 🚀 JARVIS AI - Advanced Command Suite (v2.0)

JARVIS now supports over 100+ specialized actions for deep system control, web automation, and productivity.

## 🖥️ 1. System Control & Intelligence

| Command              | Action                          | Hindi/Hinglish Example        |
| :------------------- | :------------------------------ | :---------------------------- |
| `focus_app`          | Brings specific window to front | "Chrome focus karo"           |
| `minimize_app`       | Minimizes a window              | "Calculator chhota kar do"    |
| `maximize_app`       | Maximizes a window              | "Code editor bada karo"       |
| `get_ip_address`     | Shows Local & Public IP         | "Mera IP address kya hai?"    |
| `get_network_status` | Current data usage stats        | "Internet speed/status dikha" |
| `suspend_process`    | Pauses a process (by PID)       | "Pid 1234 ko suspend karo"    |
| `resume_process`     | Resumes a paused process        | "Process resume karo"         |
| `sleep_system`       | Puts PC to sleep                | "Computer sula do"            |

## 🛡️ Security Guard

1. **Confirmation**: Dangerous actions (delete, shutdown, install) require "Haan, kar do!"
2. **Blocked Paths**: Cannot touch System32 or Program Files.
3. **Low Confidence**: JARVIS will ask "Kya tum sure ho?" if intent is ambiguous.

---

## 🧬 How to use Complex Tasks?

You can combine any of these using "and", "then", or "fir".
_Example: "Desktop pe 'Report' folder banao, fir Chrome kholo and summarize karo [URL]."_
